# TabulationX - Static Presentation Demo

This is a fully self-contained, static HTML/CSS/JS version of the **TabulationX** system. It is designed for client demonstrations and can be hosted directly on **GitHub Pages** without any backend requirements.

## ✨ Features
- **Polished Sidebar**: Professional, consistent layout with high-end typography.
- **Dynamic Dashboard**: Replicated landing page with interactive charts (Chart.js).
- **Multi-Perspective View**:
  - **Main Page**: Executive dashboard overview.
  - **Student Portal**: Perspective of a project researcher/student.
  - **Faculty Portal**: Perspective of a judge/evaluator.
- **Login Simulation**: Fully designed login portal for a complete user flow.
- **GitHub Pages Ready**: Uses relative paths and static assets for seamless deployment.

## 🚀 Deployment Instructions (GitHub Pages)

Follow these steps to host this demo on your GitHub account:

### 1. Prepare your Repository
- Go to your GitHub account and create a new repository (e.g., `tabulation-demo`).
- Clone the repository to your local machine.

### 2. Copy the Demo Folder
- Navigate to your project directory.
- Copy the entire contents of the `demo/` folder and paste them into your new repository's root.
- **Ensure `index.html` is at the root level of your new repository.**

### 3. Commit and Push
```bash
git add .
git commit -m "Initial commit - TabulationX Static Demo"
git push origin main
```

### 4. Enable GitHub Pages
- Go to your repository on **GitHub.com**.
- Click **Settings** > **Pages** (on the left sidebar).
- Under **Build and deployment**, select:
  - Source: **Deploy from a branch**
  - Branch: **main** (or your primary branch), Folder: **/(root)**
- Click **Save**.

### 5. Access your Demo
- After a minute, GitHub will provide a link like: `https://yourusername.github.io/tabulation-demo/`
- Share this link with your clients for a professional, live demonstration!
